How to Use the PDF to JPG Tool

Last Updated: October 11, 2025

1. Accessing the PDF to JPG Tool

Navigate to the Nova Tools homepage and locate the "PDF to JPG" tool card in the Tools section. Click the card, which displays a file-export icon and the description "Convert the first page of your PDF to a JPG image," to open the PDF to JPG modal.

2. Opening the PDF to JPG Modal

After clicking the tool card, a modal titled "Free PDF to JPG Converter" will appear. This modal contains an upload area for selecting a PDF file. You can close the modal by clicking the "Ã—" button in the top-right corner or clicking outside the modal.

3. Selecting a PDF File

Click the "Choose File" link in the upload area or drag and drop a PDF file into it. The name of the selected file (e.g., "Selected: document.pdf") will appear below the upload area. Only one PDF file can be selected at a time, and only PDF format is supported.

4. Converting PDF to JPG

Click the "Convert File" button to start converting the first page of the PDF to a JPG image. A progress bar will appear, showing the conversion progress. The button will be disabled during processing to prevent multiple submissions. If no file is selected, an alert will prompt you to select a PDF file.

5. Downloading the JPG

After processing, the "Convert File" button is replaced by a "Download JPG" button with a download icon. Click this button to download the JPG image, named after the input PDF (e.g., "document.jpg"). The modal remains open, allowing you to convert another PDF or close it.
