How to Use the Add Watermark to PDF Tool

Last Updated: October 11, 2025

1. Accessing the Add Watermark to PDF Tool

Navigate to the Nova Tools homepage and locate the "Add Watermark" tool card in the Tools section. Click the card, which displays a stamp icon and the description "Add text or image watermarks to your PDF documents," to open the watermark modal.

2. Opening the Watermark Modal

After clicking the tool card, a modal titled "Add Watermark to PDF" will appear. This modal contains an upload area for selecting a PDF file. You can close the modal by clicking the "Ã—" button in the top-right corner or clicking outside the modal.

3. Selecting a PDF File

Click the "Choose File" button in the upload area or drag and drop a PDF file (`.pdf`) into it. The name of the selected file (e.g., "Selected: document.pdf") will appear below the upload area. Only one PDF file can be selected, and only `.pdf` format is supported.

4. Configuring Watermark Options

Click the "Add Watermark" button to open the watermark options panel. Choose between:

  • Image Watermark: Upload an image (`.png`, `.jpg`, `.jpeg`), adjust scale (10â€“200%), and view a preview.

5. Processing the Watermark

Click the "Process" button to apply the watermark. A progress bar will show the processing status for each page. The button will be disabled during processing. If no PDF, text, or image is provided (as applicable), an alert will prompt you to correct the issue.

6. Downloading the Watermarked PDF

After processing, the "Process" button is replaced by a "Download Watermarked PDF" button with a download icon. Click it to download the watermarked PDF, named `watermarked_document.pdf`. The modal remains open for further actions.
