How to Use the Image Compressor

Last Updated: October 12, 2025

1. Accessing the Image Compressor

Visit the Nova Tools homepage and locate the "Image Compressor" tool card in the Tools section. The card displays a compression icon and the description "Compress JPG or PNG images to reduce file size." Click the card to open the compressor modal.

2. Opening the Compressor Modal

Clicking the tool card opens a modal titled "Free Image Compressor". This modal includes an upload area for selecting JPG or PNG images, a compression level selector, a file list, a progress bar, and buttons for processing and downloading. Close the modal by clicking the "Ã—" button or outside the modal.

3. Uploading Images

Click the "Choose Files" link or drag and drop JPG/PNG images into the upload area. The area highlights when dragging files. Selected files appear in a list below, showing each fileâ€™s name and original size (e.g., 1.23 MB). Duplicate files are ignored.

4. Setting Compression Level

Choose a compression level (e.g., low, medium, high) using a slider or dropdown in the modal. Higher compression reduces file size more but may lower image quality. Adjust based on your needs (e.g., medium for a balance of size and quality).

5. Managing Uploaded Files

The file list shows uploaded images with names and original sizes. Remove a file by clicking the "Ã—" icon next to it. The upload area updates to show the number of selected files (e.g., "Selected: 3 files") or the file name if only one is selected.

6. Compressing Images

Click the "Compress Files" button to start compression. A progress bar appears, showing the compression progress for each image. If no files are selected, an alert appears: "Please select JPG/PNG images to compress."

7. Viewing Compression Progress

During compression, a progress bar fills from 0% to 100% as each image is processed. If an error occurs (e.g., invalid file format), an alert displays: "Error compressing images: [error message]." The modal resets to allow retrying.

8. Downloading Compressed Images

After compression, a "Download Result" button with a pulsing animation appears. Click it to download the compressed images, either individually or as a ZIP file (e.g., "compressed_images.zip"). The file list shows original and compressed sizes for comparison.
