How to Use the Image Resizer

Last Updated: October 12, 2025

1. Accessing the Image Resizer

Visit the Nova Tools homepage and locate the "Image Resizer" tool card in the Tools section. The card displays a resize icon and the description "Adjust the dimensions of your image." Click the card to open the resizer modal.

2. Opening the Resizer Modal

Clicking the tool card opens a modal titled "Free Image Resizer". This modal includes an upload area for selecting a JPG or PNG image, input fields for target width and height, a file list, a progress bar, and buttons for resizing and downloading. Close the modal by clicking the "Ã—" button or outside the modal.

3. Uploading an Image

Click the "Choose File" link or drag and drop a JPG or PNG image into the upload area. The area highlights when dragging a file. The selected file appears in a list below, showing its name and original size (e.g., 1.23 MB). Only one image can be uploaded at a time. If an invalid file is selected, an alert appears: "Yo, that's not a valid image! Pick a JPG or PNG and try again."

4. Setting Target Dimensions

Enter the desired width and height in pixels in the provided input fields (e.g., 800 for width, 600 for height). If no dimensions are entered, the tool defaults to 50% of the original image's dimensions. The target dimensions must be smaller than the original image size, or an alert appears: "Target dimensions must be smaller than the original image (WIDTHxHEIGHT). Shrink it down!"

5. Managing the Uploaded File

The file list shows the uploaded image with its name and original size. Remove the file by clicking the "Ã—" icon next to it. The upload area updates to show the file name or "No file chosen" if no file is selected.

6. Resizing the Image

Click the "Resize Image" button to start resizing. A progress bar appears, showing the resizing progress. If no file is selected, an alert appears: "Yo, pick an image first, dude!" If the image is invalid, an alert appears: "Something went wrong resizing the image. Make sure itâ€™s a legit JPG or PNG and try again."

7. Viewing Resizing Progress

During resizing, a progress bar fills from 0% to 100% as the image is processed. The bar reaches 50% when the image is drawn to a canvas and 100% when resizing is complete. If an error occurs, the modal resets to allow retrying.

8. Downloading the Resized Image

After resizing, a "Download Resized Image" button with a pulsing animation appears. Click it to download the resized image, named "resized_image" with the original file extension (e.g., resized_image.jpg). The modal resets after the download.
