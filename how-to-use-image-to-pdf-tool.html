How to Use the Image to PDF Tool

Last Updated: October 11, 2025

1. Accessing the Image to PDF Tool

Navigate to the Nova Tools homepage and locate the "Image to PDF" tool card in the Tools section. Click the card, which displays a file-image icon and the description "Convert JPG or PNG images to a PDF document," to open the Image to PDF modal.

2. Opening the Image to PDF Modal

After clicking the tool card, a modal titled "Free Image to PDF Converter" will appear. This modal contains an upload area for selecting JPG or PNG images. You can close the modal by clicking the "Ã—" button in the top-right corner or clicking outside the modal.

3. Selecting JPG or PNG Images

Click the "Choose Files" link in the upload area or drag and drop one or more JPG or PNG images into it. The number of selected files (e.g., "Selected: 3 files") or the name of a single file will appear below the upload area. Only JPG and PNG formats are supported.

4. Converting Images to PDF

Click the "Process Files" button to start converting the images to a PDF. A progress bar will appear, showing the conversion progress for each image. The button will be disabled during processing to prevent multiple submissions. If no files are selected, an alert will prompt you to select images.

5. Downloading the PDF

After processing, the "Process Files" button is replaced by a "Download Result" button with a download icon. Click this button to download the PDF file, named "converted_document.pdf", containing all selected images as separate pages. The modal remains open, allowing you to convert more images or close it.
