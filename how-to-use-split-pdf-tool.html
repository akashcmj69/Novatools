How to Use the Split PDF Tool

Last Updated: October 11, 2025

1. Accessing the Split PDF Tool

Navigate to the Nova Tools homepage and locate the "Split PDF" tool card in the Tools section. Click the card, which displays a cut icon and the description "Split a PDF into individual pages or by custom ranges," to open the Split PDF modal.

2. Opening the Split PDF Modal

After clicking the tool card, a modal titled "Free PDF Splitter" will appear. This modal contains an upload area for selecting a PDF file and options for splitting. You can close the modal by clicking the "×" button in the top-right corner or clicking outside the modal.

3. Selecting a PDF File

Click the "Choose File" link in the upload area or drag and drop a single PDF file into it. The name and size of the selected file will appear in a list below the upload area, and the upload area will highlight, showing "Selected: [file name]". Only one PDF file can be selected at a time.

4. Splitting the PDF File

Click the "Process File" button to start splitting. A progress bar will appear, showing the processing progress. The button will be disabled during processing to prevent multiple submissions. If no file is selected or the page range is invalid, an alert will prompt you to correct the input.

5. Downloading the Split PDF

After processing, the "Process File" button is replaced by a "Download Results" button with a download icon. Click this button to download the split PDF file, named "split_document.pdf". The modal remains open, allowing you to split another file or close it.
