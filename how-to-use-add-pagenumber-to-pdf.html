How to Use the Add Page Number to PDF Tool

Last Updated: October 11, 2025

1. Accessing the Add Page Number to PDF Tool

Navigate to the Nova Tools homepage and locate the "Add Page Number" tool card in the Tools section. Click the card, which displays a numbered list icon and the description "Add customizable page numbers to your PDF documents," to open the page number modal.

2. Opening the Page Number Modal

After clicking the tool card, a modal titled "Add Page Numbers to PDF" will appear. This modal contains an upload area for selecting a PDF file. You can close the modal by clicking the "×" button in the top-right corner or clicking outside the modal.

3. Selecting a PDF File

Click the "Choose PDF File" button in the upload area or drag and drop a PDF file (`.pdf`) into it. The selected file’s name (e.g., "Selected: document.pdf") and details (name, size) will appear in the file list below. Only one PDF file can be selected; choosing a new file replaces the previous one.

4. Configuring Page Number Options

Click the "Add Page Numbers" button to open the options modal. Configure the following:

  • Page Number Position: Choose from Bottom Right, Bottom Left, Top Right, or Top Left.
  • Font Size: Set the size (8–24 pt, default 12 pt).
  • Font Color: Select a color using the color picker (default black).
  • Start Page Number: Specify the starting number (minimum 1, default 1).

5. Processing the Page Numbers

Click the "Process" button to add page numbers. A progress bar will show the processing status for each page. The button will be disabled during processing. If no PDF is selected, an alert will prompt you to upload a file.

6. Downloading the Numbered PDF

After processing, the "Process" button is replaced by a "Download Result" button with a download icon. Click it to download the numbered PDF, named `numbered_pdf.pdf`. The modal remains open for further actions.
